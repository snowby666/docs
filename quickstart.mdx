## Make Your First Request

Choose your preferred programming language and start building:

<CodeGroup>

```python Python
from openai import OpenAI

# Initialize the client
client = OpenAI(
    base_url="https://api.electronhub.top/v1",
    api_key="ek-your-api-key-here"
)

# Make a chat completion request
response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Hello! How are you?"}
    ]
)

print(response.choices[0].message.content)
```


```javascript Node.js
import { OpenAI } from 'openai';

// Initialize the client
const client = new OpenAI({
    baseURL: 'https://api.electronhub.top/v1',
    apiKey: 'ek-your-api-key-here'
});

// Make a chat completion request
async function main() {
    const response = await client.chat.completions.create({
        model: 'gpt-4o',
        messages: [
            { role: 'system', content: 'You are a helpful assistant.' },
            { role: 'user', content: 'Hello! How are you?' }
        ]
    });
    
    console.log(response.choices[0].message.content);
}

main();
```


```bash cURL
curl https://api.electronhub.top/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ek-your-api-key-here" \
  -d '{
    "model": "gpt-4o",
    "messages": [
      {
        "role": "system",
        "content": "You are a helpful assistant."
      },
      {
        "role": "user", 
        "content": "Hello! How are you?"
      }
    ]
  }'
```

## Enable Streaming

For real-time responses, enable streaming:

<CodeGroup>

```python Python Streaming
stream = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "user", "content": "Write a short poem about space"}
    ],
    stream=True
)

for chunk in stream:
    if chunk.choices[0].delta.content is not None:
        print(chunk.choices[0].delta.content, end="")
```


```javascript Node.js Streaming
const stream = await client.chat.completions.create({
    model: 'gpt-4o',
    messages: [
        { role: 'user', content: 'Write a short poem about space' }
    ],
    stream: true
});

for await (const chunk of stream) {
    if (chunk.choices[0]?.delta?.content) {
        process.stdout.write(chunk.choices[0].delta.content);
    }
}
```

</CodeGroup>

## Function Calling

Enable your AI to use external tools:

<CodeGroup>

```python Python Functions
tools = [
    {
        "type": "function",
        "function": {
            "name": "get_weather",
            "description": "Get current weather for a location",
            "parameters": {
                "type": "object",
                "properties": {
                    "location": {
                        "type": "string",
                        "description": "The city and state"
                    }
                },
                "required": ["location"]
            }
        }
    }
]

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "user", "content": "What's the weather in San Francisco?"}
    ],
    tools=tools,
    tool_choice="auto"
)
```


```javascript Node.js Functions
const tools = [
    {
        type: 'function',
        function: {
            name: 'get_weather',
            description: 'Get current weather for a location',
            parameters: {
                type: 'object',
                properties: {
                    location: {
                        type: 'string',
                        description: 'The city and state'
                    }
                },
                required: ['location']
            }
        }
    }
];

const response = await client.chat.completions.create({
    model: 'gpt-4o',
    messages: [
        { role: 'user', content: "What's the weather in San Francisco?" }
    ],
    tools: tools,
    tool_choice: 'auto'
});
```

</CodeGroup>

## Generate Images

Create images from text descriptions:

<CodeGroup>

```python Python Images
response = client.images.generate(
    model="dall-e-3",
    prompt="A futuristic city skyline at sunset with flying cars",
    size="1024x1024",
    quality="standard",
    n=1
)

image_url = response.data[0].url
print(f"Generated image: {image_url}")
```


```javascript Node.js Images
const response = await client.images.generate({
    model: 'dall-e-3',
    prompt: 'A futuristic city skyline at sunset with flying cars',
    size: '1024x1024',
    quality: 'standard',
    n: 1
});

const imageUrl = response.data[0].url;
console.log(`Generated image: ${imageUrl}`);
```
