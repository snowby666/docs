---
title: "Get Model"
openapi: "GET /models/{model}"
---

Retrieve information about a specific model.

## Path Parameters

The model ID is passed as a path parameter in the URL.

## Response

Returns detailed information about the specified model including its capabilities, context length, and other metadata.

## Example Response

```json
{
  "id": "gpt-4o",
  "object": "model",
  "owned_by": "openai",
  "created": 1677610602,
  "context_length": 128000,
  "capabilities": {
    "chat": true,
    "vision": true,
    "function_calling": true,
    "streaming": true
  }
}
```

## Common Models

### GPT Models
- `gpt-4o` - Latest GPT-4 model with vision
- `gpt-4` - Standard GPT-4 model
- `gpt-3.5-turbo` - Fast and efficient model

### Claude Models  
- `claude-3-5-sonnet-20241022` - Latest Claude model
- `claude-3-haiku-20240307` - Fastest Claude model

### Gemini Models
- `gemini-pro` - Google's flagship model
- `gemini-pro-vision` - Gemini with vision capabilities

## Error Responses

Returns a 404 error if the model is not found:

```json
{
  "error": {
    "message": "Model not found",
    "type": "invalid_request_error",
    "code": "model_not_found"
  }
}
```
