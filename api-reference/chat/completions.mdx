Create a chat completion using the OpenAI-compatible format. This endpoint supports 200\+ models including GPT-4, Claude, Gemini, and more through a unified interface.

## Features

- **200\+ Models**: Access models from OpenAI, Anthropic, Google, Meta, and more
- **Streaming**: Real-time response streaming for interactive applications
- **Function Calling**: Enable AI to use external tools and APIs
- **Vision**: Process images alongside text (vision-enabled models)
- **Advanced Reasoning**: Thinking modes for complex problem solving
- **Web Search**: Enable AI to search the web for current information

## Examples

### Basic Chat Completion

<CodeGroup>

```python Python
from openai import OpenAI

client = OpenAI(
    base_url="https://api.electronhub.top/v1",
    api_key="ek-your-api-key"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Hello! How are you?"}
    ]
)

print(response.choices[0].message.content)
```


```javascript Node.js
import { OpenAI } from 'openai';

const client = new OpenAI({
    baseURL: 'https://api.electronhub.top/v1',
    apiKey: 'ek-your-api-key'
});

const response = await client.chat.completions.create({
    model: 'gpt-4o',
    messages: [
        { role: 'system', content: 'You are a helpful assistant.' },
        { role: 'user', content: 'Hello! How are you?' }
    ]
});

console.log(response.choices[0].message.content);
```


```bash cURL
curl https://api.electronhub.top/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ek-your-api-key" \
  -d '{
    "model": "gpt-4o",
    "messages": [
      {
        "role": "system",
        "content": "You are a helpful assistant."
      },
      {
        "role": "user",
        "content": "Hello! How are you?"
      }
    ]
  }'
```

</CodeGroup>

### Streaming Responses

<CodeGroup>

```python Python
stream = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "user", "content": "Write a short poem about space"}
    ],
    stream=True
)

for chunk in stream:
    if chunk.choices[0].delta.content is not None:
        print(chunk.choices[0].delta.content, end="", flush=True)
```


```javascript Node.js
const stream = await client.chat.completions.create({
    model: 'gpt-4o',
    messages: [
        { role: 'user', content: 'Write a short poem about space' }
    ],
    stream: true
});

for await (const chunk of stream) {
    if (chunk.choices[0]?.delta?.content) {
        process.stdout.write(chunk.choices[0].delta.content);
    }
}
```


```bash cURL
curl https://api.electronhub.top/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ek-your-api-key" \
  -d '{
    "model": "gpt-4o",
    "messages": [
      {
        "role": "user",
        "content": "Write a short poem about space"
      }
    ],
    "stream": true
  }'
```

</CodeGroup>

### Function Calling

<CodeGroup>

```python Python
tools = [
    {
        "type": "function",
        "function": {
            "name": "get_weather",
            "description": "Get current weather for a location",
            "parameters": {
                "type": "object",
                "properties": {
                    "location": {
                        "type": "string",
                        "description": "The city and state, e.g. San Francisco, CA"
                    }
                },
                "required": ["location"]
            }
        }
    }
]

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "user", "content": "What's the weather in San Francisco?"}
    ],
    tools=tools,
    tool_choice="auto"
)
```


```javascript Node.js
const tools = [
    {
        type: 'function',
        function: {
            name: 'get_weather',
            description: 'Get current weather for a location',
            parameters: {
                type: 'object',
                properties: {
                    location: {
                        type: 'string',
                        description: 'The city and state, e.g. San Francisco, CA'
                    }
                },
                required: ['location']
            }
        }
    }
];

const response = await client.chat.completions.create({
    model: 'gpt-4o',
    messages: [
        { role: 'user', content: "What's the weather in San Francisco?" }
    ],
    tools: tools,
    tool_choice: 'auto'
});
```
