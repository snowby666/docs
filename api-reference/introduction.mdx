## What is Electron Hub?

Electron Hub is a unified API platform that provides seamless access to over **200 cutting-edge AI models** from leading providers including OpenAI, Anthropic, Google, Meta, Mistral, and many more. Our platform acts as a single gateway, allowing developers to integrate multiple AI capabilities without managing separate APIs for each provider.

## API Formats

Electron Hub supports multiple API formats for maximum flexibility:

### OpenAI Compatible Format
Perfect for existing applications and maximum compatibility:

```python
from openai import OpenAI

client = OpenAI(
    base_url="https://api.electronhub.ai/v1",
    api_key="ek-your-api-key"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}]
)
```

### Anthropic Native Format
Access advanced Claude features with native Anthropic API:

```python
import anthropic

client = anthropic.Anthropic(
    base_url="https://api.electronhub.ai/v1",
    api_key="ek-your-api-key"
)

response = client.messages.create(
    model="claude-3-5-sonnet-20241022",
    max_tokens=1000,
    messages=[{"role": "user", "content": "Hello!"}]
)
```
