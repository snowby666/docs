## What is Electron Hub API?

Electron Hub provides a unified API gateway to access over **200 cutting-edge AI models** from leading providers like OpenAI, Anthropic, Google, Meta, and more. Whether you're building chatbots, generating images, or creating embeddings, our platform offers seamless integration with multiple AI capabilities.

## Supported Capabilities

<AccordionGroup>
  <Accordion title="Chat Completions" icon="comments">
    Generate human-like text responses with support for:
    - **Streaming responses** for real-time applications
    - **Function calling** for tool integration
    - **Vision capabilities** for image understanding
    - **Advanced reasoning** with thinking modes
    
    Compatible with OpenAI and Anthropic formats.
  </Accordion>

  <Accordion title="Image Generation" icon="palette">
    Create stunning images from text prompts:
    - **DALL-E 3** and **DALL-E 2** models
    - **Stable Diffusion** variants
    - **Midjourney-style** models
    - Custom sizes and quality settings
  </Accordion>

  <Accordion title="Embeddings" icon="vector-square">
    Convert text into numerical vectors for:
    - **Semantic search** applications
    - **Recommendation systems**
    - **Clustering and classification**
    - Multiple embedding dimensions
  </Accordion>

  <Accordion title="Content Moderation" icon="shield-check">
    Ensure safe content with:
    - **Hate speech detection**
    - **Violence screening**
    - **Sexual content filtering**
    - **Self-harm prevention**
  </Accordion>
</AccordionGroup>
