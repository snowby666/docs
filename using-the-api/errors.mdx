## API errors

| Code                                                            | Overview                                                                                                                                                     |
| :-------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **401** - Invalid Authentication                                | **Cause:** Invalid Authentication&#xA;**Solution:** Ensure the correct API key are being used.                                                               |
| **401** - Incorrect API key provided                            | **Cause:** The requesting API key is not correct.&#xA;**Solution:** Ensure the API key used is correct, clear your browser cache, or generate a new one.     |
| **429** - Rate limit reached for requests                       | **Cause:** You are sending requests too quickly.&#xA;**Solution:** Pace your requests. [Check your rate limits](https://platform.monica.im/dashboard/limit). |
| **4**02 - Insufficient balance                                  | **Cause:** You have run out of balance.&#xA;**Solution:** Wait till next daily reset or upgrade your plan                                                    |
| 402 - Premium model requires a subscription                     |                                                                                                                                                              |
| **500** - The server had an error while processing your request |                                                                                                                                                              |
| **503** - Unavailable service                                   |                                                                                                                                                              |
