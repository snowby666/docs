---
title: "Rate limits"
description: "Electron Hub API has a rate limit system to prevent abusive activities and ensure fair usage among all users, and this document lists the default rate limits for each subscription plans"
---

<Tip>
  You can check your actual rate limits anytime on [the API Keys section](https://platform.monica.im/dashboard/limits) in the console.
</Tip>

| Subscription Plan | Requests per minute (RPM) |
| ----------------- | ------------------------- |
| Free              | 7 RPM                     |
| Starter           | 10 RPM                    |
| Plus              | 20 RPM                    |
| Pro               | 40 RPM                    |
| Business          | 60 RPM                    |
| Enterprise        | 100 RPM                   |
